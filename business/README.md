# Modelo de negocio - NeuroFarmac

Esta carpeta contiene el análisis financiero y modelo de negocio para la plataforma.

## Contenido planificado

- `cost_analysis.md` — análisis de costos de componentes y operación.
- `pricing.md` — estrategia de precios para laboratorios y farmacéuticas.
- `market.md` — mercado objetivo: laboratorios de neurociencia, compañías farmacéuticas.
- `roi.md` — retorno de inversión para el usuario final.

## Costo estimado

| Componente | Costo estimado (USD) |
|------------|----------------------|
| Chip microfluídico | 10–50 |
| Membrana BHE | 5–20 |
| MEA (256 electrodos) | 500–2,000 |
| Bomba de flujo | 200–500 |
| Organoides (cultivo) | 100–500 por lote |
| Electrónica de lectura | 500–2,000 |
| **Total sistema** | **~1,500–5,000** |

## Modelos de negocio

1. **Venta del sistema** — a laboratorios de neurociencia (equipo completo).
2. **Kit de componentes** — para laboratorios que ya tienen MEA o bomba.
3. **Servicio de análisis** — el laboratorio envía fármacos, NeuroFarmac entrega datos.
4. **Licenciamiento** — a compañías farmacéuticas para uso interno.
