# Hardware - NeuroFarmac

Esta carpeta contiene los esquemas y especificaciones del chip microfluídico, la barrera hematoencefálica (BHE) y el arreglo de microelectrodos (MEA).

## Contenido planificado

- `chip_microfluidos/` — diseño del chip de PDMS o vidrio (canales de 100–200 µm ancho, 50–100 µm alto).
- `bhe_membrane.md` — especificaciones de la membrana para co-cultivo de endotelio, pericitos y astrocitos.
- `mea_specs.md` — especificaciones del arreglo de microelectrodos (256–1024 electrodos, espaciado 20–50 µm).
- `fluidic_control.md` — sistema de flujo pulsátil (0.5–5 µL/min).

## Componentes principales

| Componente | Especificación | Función |
|------------|----------------|---------|
| Chip microfluídico | PDMS o vidrio grabado | Sostiene flujo y células |
| Membrana BHE | Polímero poroso | Co-cultivo de células de barrera |
| MEA | 256–1024 electrodos, 20–50 µm espaciado | Monitoreo eléctrico |
| Bomba de flujo | Pulsátil, 0.5–5 µL/min | Simula circulación sanguínea |
