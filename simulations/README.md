# Simulaciones - NeuroFarmac

Esta carpeta contiene modelos computacionales para validar el comportamiento del sistema.

## Contenido planificado

- `fluid_dynamics.sim` — simulación de flujo en los microcanales (ANSYS Fluent).
- `bhe_permeability.sim` — modelo de permeabilidad de la barrera hematoencefálica.
- `neural_activity.sim` — simulación de actividad neuronal captada por los MEA.
- `drug_diffusion.sim` — difusión del fármaco desde el flujo hasta el organoide.

## Parámetros clave

- Flujo: 0.5–5 µL/min, pulsátil.
- Temperatura: 37°C.
- CO₂: 5%.
- Organoide: 2–4 mm diámetro.
- MEA: 256–1024 electrodos, muestreo >20 kHz.
